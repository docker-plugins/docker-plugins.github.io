Unofficial Plugins Demos

Experimental plugins demos with Flocker, Weave, Swarm & Compose

Here are some EXPERIMENTAL demos using Flocker and Weave as Docker plugins...

This website and repo will allow you to play with a Vagrant cluster in the following configuration.

$ git clone https://github.com/dockercon-plugins-2015/demo
$ cd demo
$ make -C vagrantrunner

Who's behind this?

This repo and landing page has been put together as a joint project by the nice folks at ClusterHQ and Weave.

In terms of building the Docker plugin functionality, we are proud to have worked closely with the Docker core team to build the first ever Docker plugins.

See #13161 and #13441 etc.
