ClusterHQ and Weaveworks are proud to have worked closely with the Docker core team to deliver plugins functionality in Docker engine, which is now available in the experimental channel of Docker.

Try the demo yourself using Vagrant and VirtualBox

Prerequisites: Vagrant 1.7.2 
$ git clone https://github.com/plugins-demo-2015/demo
$ cd demo
$ make -C vagrantrunner

Warning: the Vagrant box is 1.3GB.

Then follow the instructions in the README.
