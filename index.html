Unofficial Plugin Demos

Experimental plugin demos with Flocker, Weave, Docker Swarm & Docker Compose

Here are some experimental demos using Flocker and Weave as Docker plugins with Docker Swarm and Docker Compose

This website and repo will allow you to play with a Vagrant cluster in the following configuration.

Prerequisites for Linux or OS X

  • Vagrant 1.7.2
  • flocker-cli tool
  • unofficial flocker installer (contains flocker-plugin-install tool)
$ git clone https://github.com/plugin-demos-2015/demo
$ cd demo
$ make -C vagrantrunner

Who's behind this?

This repo and landing page has been put together as a joint project by the nice folks at ClusterHQ and Weaveworks.

In terms of building the Docker plugin functionality, we are proud to have worked closely with the Docker core team to build the first ever Docker plugins.

See #13161 and #13441 etc.
